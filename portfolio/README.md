# Pragati Portfolio

Personal portfolio website built to showcase my work, skills, and experience as a developer.

## About This Project

This is my personal portfolio application, created with a modern frontend stack and interactive UI sections.
It includes:

- Hero section with animated visual effects
- Projects showcase
- Experience timeline
- Testimonials and client highlights
- Contact-focused call to action

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Three.js / React Three Fiber

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Open in browser:

```text
http://localhost:3000
```

## Production Build

```bash
npm run build
npm run start
```

## Deployment

This project is configured for Netlify static deployment.

- Build command: `npm run build`
- Publish directory: `out`

Netlify settings are defined in `netlify.toml`.

## Project Structure

```text
.
|- app/
|- components/
|- data/
|- lib/
|- public/
|- netlify.toml
|- next.config.mjs
|- package.json
```

## Customization

- Update content data in `data/index.ts`
- Modify section components in `components/`
- Adjust global styling in `app/globals.css`
